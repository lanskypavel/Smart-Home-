Detailed Description

Основной класс главного окна приложения

Предоставляет интерфейс для управления устройствами умного дома:

  • Кондиционером
  • Шторами
  • Чайником
  • Освещением
  • Розетками А также включает медиаплеер для воспроизведения музыки.
